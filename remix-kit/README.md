# Stratara Remix Kit

This folder holds files prepared outside the ISO before copying them into the Cubic build VM.

The first ISO should stay simple:

```text
branding/   Logos, color references, wallpaper notes
calamares/  Installer branding and slides
kde/        Plasma defaults, color scheme, look-and-feel notes
plymouth/   Boot splash theme notes
sddm/       Login screen theme notes
```

Use this kit as the staging area for Stratara OS v0.1 assets.
